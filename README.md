# 🛒 SecureCart – JWT Protected Shopping Service

SecureCart is a backend e-commerce application built using **Java** and **Spring Boot**. It provides secure REST APIs for managing products, categories, carts, and orders, with **JWT-based authentication** using Spring Security.

---

## 🚀 Features

### 🔐 Authentication & Authorization
- User registration and login
- JWT-based authentication
- Role-based access control (User/Admin)
- Secure REST APIs

### 📦 Product & Category Management
- CRUD operations for products and categories
- Image upload and mapping
- One-to-Many and Many-to-One relationships

### 🛒 Shopping Cart
- Add items to cart
- Remove items from cart
- Update item quantities
- View cart details

### 📑 Order Management
- Convert cart into order
- Store purchase history
- Retrieve user-specific orders
